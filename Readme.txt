Description:

Scheduler is a full-stack web application that is designed to record and manage important events or tasks,
unlike a traditional calendar, it focuses solely on displaying entries that the user has actively recorded.

====================================================================

Technology Stack:

Front-end:
ReactJS
Typescript

Back-end:
SpringBoot
