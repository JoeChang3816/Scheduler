====================================================================  

Description:  

Scheduler is a full-stack web application that is designed to record and manage important events or tasks,  
unlike a traditional calendar, it focuses solely on displaying entries that the user has actively recorded.

====================================================================  

Technology Stack:  

Front-end:  
ReactJS    
Typescript

Back-end:  
SpringBoot     
MySQL  

====================================================================  

Purpose:  

This project was created to demonstrate my programming skills,  
in addition to demonstrate my self-taught ability as I was learning ReactJS, Typescript, and SpringBoot during the creation of this project.
